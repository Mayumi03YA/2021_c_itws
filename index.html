How to use

キーワードをもとに本を検索すると、書影が表示されます。

書影をクリックすると書誌情報が表示されます。

Data Source

以下のAPIを使用しています

  • CiNii Books : 書誌情報
  • NDL Search : 書影（JPRO）
  • OpenBD : 内容紹介
  • Rakuten Books : レビュー

以下のオープンデータを使用しています

  • NDLSH(TAB区切りテキスト形式データ)
